try the development version

Development version can be checked out from the GIT repository.

For command-line subversion client, use the following command:

git clone https://github.com/ariya/FastLZ.git

After that, see instructions on how to compile it or read file README.TXT.

Note that the development version is considered unstable and may contain bugs. Please be careful when using it.
